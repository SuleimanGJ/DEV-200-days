# 🚀 Full-Stack Todo App 
- A progressive milestone to build a **Full-Stack Todo Application** from beginner level to production-ready. This project helps you learn backend, frontend, authentication, database and deployment step by step.

---

## 📌 Overview

This repository is not just a Todo App -- it is a **learning journey** devided into multiple levels:

- Basic CRUD App
- Intermediate Backend Practice
- Authentication System
- Databasi integration (MERN)
- Advanced Production Features
- Pro-Level SaaS App


---


# Version 1

## Features
- Add todo
- View todo
- Delete todo
- Mark todo as completed

## Backend
- REST API:
 - `GET /todos`
 - `POST /todos`
 - `DELETE /todos/:id`
- Data stored in:
 - in-memory array OR JSON file


## Frontend
- Form input
- Todo list rendering
- `fetch()` API calls
